# mtm6201-final

**Tell us about your process, challenges you faced during development and how you overcame those changes. What have you learned by creating this project?**

I really wanted to get the basic structure of the website, so I used a mixed of html and bootstrap to build the blueprint of the site. Once I was satisfied it was time to write the css. I will admit it did take some time to figure out, especially when I got to the media queries. Re-arranging the containers and spacing the text based on the type of device was viewing it deemed itself quite challenging. Some challenges i'd expierence from this project was trying to make my website as close to the prototype as possible. There were some things I had to remove and others I had to add in order to maintain the functionality of the site and ensure it still held the same vision. Through this project i've learn how to convert a vision into a full website, and bring it to life.
